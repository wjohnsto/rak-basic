    Features demonstrated

    Task Lifecycle API request creates a task quickly, then a worker handles execution out of band.
    SSE Streaming The UI listens to /tasks/{id}/stream while the task runs.
    Pause And Resume Gated prompts call request_input(), then continue after submit_input().
    Protocol Surface See /protocol for quick A2A, ACP, and MCP examples for this demo.
